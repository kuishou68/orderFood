# 前言
外卖平台移动端项目，包含登录注册、商铺列表、订单详情、积分、地址、搜索、当前城市、余额、红包、服务中心等模块。

## 技术栈

vue2 + vuex + vue-router + webpack + ES6/7 + fetch + sass + flex + svg


## 项目运行

```

cd orderFood

npm install 或 yarn(推荐)

npm run dev

```
## 关于接口

此项目的所有接口数据都来源于配套的后台系统，[接口文档](https://api.52kfw.cn/api.html)。
